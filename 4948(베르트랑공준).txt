// 나는 이 문제의 핵심을 "에라토스테네스의 체를 알고 있고 그것을 응용할 수 있는가?"라고 생각한다.
// 따라서 벡터 메모리를 2배로 해주고 for문 i의 최대범위, 그리고 for문 안에 while문 탐색 범위 또한 2배로 해주었다.  
#include <bits/stdc++.h>

using namespace std;

int n;

int main() {
        ios::sync_with_stdio(false);
        cin.tie(NULL);

        while(1) {
                cin >> n;
                if (n == 0) return 0;
                vector<int> arr(2*(n + 1),true);
                for (int i = 2; i <= (int) sqrt(2*n); i++) {
                        if (arr[i] == true) {
                                int j = 2;
                                while (i * j <= 2*n) {
                                        arr[i * j] = false;
                                        j += 1;
                                }
                        }
                }
                int cnt = 0;
                for (int i = n+1; i <= 2*n; i ++) {
                        if (arr[i]) cnt ++;
                }
                cout << cnt << '\n';
        }

}
